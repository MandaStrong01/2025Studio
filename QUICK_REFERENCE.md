# QUICK REFERENCE GUIDE

Fast reference for MandaStrong Studio features

## AI Tool Workflow

### Creating Content - Two Methods

**Method 1: Upload**
1. Click tool → Upload
2. Choose file
3. Process & Upload
4. Use **Add to Timeline** button

**Method 2: Create with AI**
1. Click tool → Create with AI
2. Enter prompt (up to 2000 chars)
3. Click **+ button** to add reference file (optional)
4. Choose style & quality
5. Generate
6. Use **Add to Timeline** button

## Key Features

### Add to Timeline Button
- Appears after upload or generation
- Instantly adds media to track 1
- Requires active project
- Takes you to Timeline Editor

### Reference File Upload
- Click **+** icon in Create with AI mode
- Upload image or video as reference
- Helps guide AI generation
- Optional but recommended for style consistency

### Three Action Buttons
After any creation:
1. **Add to Timeline** (green) - Direct to project
2. **View in Media Library** - Opens library
3. **Generate/Upload Another** - Create more
