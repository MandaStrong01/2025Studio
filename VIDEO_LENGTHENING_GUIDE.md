# How to Lengthen Videos to 180 Minutes

## Default Settings

The Video Studio is now configured with:
- **Target Duration: 180 minutes** (default)
- **Trim Start: 0 seconds** (automatically set when video loads)
- **Trim End: Full video duration** (automatically set when video loads)

No manual configuration needed unless you want different values.

## Video Studio Method

### Steps to Set Target Duration to 180 Minutes:

1. **Navigate to Video Studio**
   - From the main menu, click "Video Studio" or navigate to `/video-studio`

2. **Verify Default Settings**
   - The target duration is **automatically set to 180 minutes** by default
   - No manual adjustment needed unless you want a different duration

3. **Select Your Video**
   - Choose a video from your uploaded videos in the "Your Videos" section
   - The video will load in the preview player
   - Trim points will automatically adjust to the video's full length

4. **Adjust Target Duration (Optional)**
   - Open the "Trim & Duration" section if needed
   - The "Target Duration" slider is already set to **180 minutes**
   - Adjust only if you want a different duration
   - The display shows: "Target Duration: 180 minutes"

5. **Set Trim Points**
   - **Start Time**: Use the "Start" slider to set when the video should begin (default: 0:00)
   - **End Time**: Use the "End" slider to set when the video should end
   - The sliders now show actual time values (e.g., 0:00, 1:30) instead of percentages

6. **Save Your Project**
   - Click the "Save Project" button to save your settings
   - Your settings are saved to the database

7. **Export the Video**
   - Click "Export Video" when ready
   - The export dialog will show all your settings including:
     - Trim range (start to end time)
     - Target duration (180 minutes)
     - Effects applied (brightness, contrast, speed, etc.)

## Tutorial Page Method

### Steps to Set Movie Duration to 180 Minutes:

1. **Navigate to Tutorials Page**
   - From the main menu, go to "Tutorials & Learning Center"
   - Or navigate directly to `/tutorials`

2. **Use Movie Duration Slider**
   - Scroll to the bottom of the page
   - Find the "Movie Duration" section with a large slider
   - The slider ranges from **0 to 180 minutes**

3. **Set Duration**
   - Drag the slider all the way to the right
   - The large display will show: **180 MINUTES**
   - The slider provides visual feedback with a purple gradient

## Troubleshooting

### Video Studio Issues:
- **Trim not saving**: Ensure video is fully loaded before setting trim points
- **Export failing**: Check that trim end time is greater than start time
- **Duration reset**: Click "Reset" button to restore default settings based on video length

### Tutorial Page Issues:
- **Slider not responding**: Refresh the page
- **Duration not persisting**: Ensure you're logged in with a valid subscription
- **Tutorials not loading**: Check internet connection and authentication status

## Best Practices

1. **Always load a video first** before adjusting trim settings
2. **Save your project frequently** to prevent losing settings
3. **Test with shorter durations** before committing to 180-minute exports
4. **Use the tutorials page** to understand all features before editing
5. **Check subscription status** to ensure access to all features
