# HOW TO USE MANDASTRONG STUDIO

Complete guide to using all features of MandaStrong Studio

## Getting Started

### 1. Registration & Login
1. Navigate to **Page 3** (Login/Register)
2. Enter your email and create a password
3. Click **Register** to create your account
4. You'll be automatically logged in after registration

### 2. Choose Your Subscription Plan
After registration, you'll see three subscription options:
- **Basic ($10/mo)**: 720p export, 5GB storage, AI tools access
- **Pro ($20/mo)**: 1080p export, 50GB storage, advanced AI tools
- **Studio ($30/mo)**: 4K export, unlimited storage, exclusive features

Select a plan to unlock full access to AI generation features.

## Using AI Tools (Pages 4-9)

### Finding the Right Tool
- Browse through **720 AI tools** across 6 pages
- Each page has 120 tools organized by category
- Use the search bar to quickly find specific tools
- Click any tool to launch its workspace

### Tool Workspace: Two Ways to Create

When you click on any AI tool, you'll see two options:

#### Option 1: Upload Existing Files
Perfect for editing, enhancing, or transforming existing media.

1. Click **Upload**
2. Choose your file (images, videos, audio, or documents)
3. Click **Process & Upload**
4. Once uploaded, you'll see three action buttons:
   - **Add to Timeline** - Instantly adds the file to your current project
   - **View in Media Library** - Opens your media library
   - **Upload Another** - Process additional files

#### Option 2: Create with AI
Generate brand new content from scratch using AI.

1. Click **Create with AI**
2. **Enter Your Prompt**: Describe what you want to create in detail
   - Use the example prompts for inspiration
   - Character limit: 2000 characters
3. **Add Reference Files (Optional)**: Click the **+ button** to upload multiple reference images or videos
   - Upload as many reference files as you need
   - This helps guide the AI to match your desired style
   - Accepts images and videos
   - Shows file size for each file
   - Remove individual files with the Remove button
4. **Choose Settings**:
   - **Style**: Professional, Creative, Minimalist, Artistic, or Realistic
   - **Quality**: Standard, High, or Ultra
5. Click **Generate with AI**
6. Once generated, you'll see three action buttons:
   - **Add to Timeline** - Instantly adds to your current project
   - **View in Media Library** - Opens your media library
   - **Generate Another** - Create more content

### Tips for Better AI Generation
- Be specific and detailed in your prompts
- Include style, lighting, mood, and composition details
- Upload multiple reference files to guide the AI's output from different angles
- Mix reference images and videos for better context
- Experiment with different styles and quality settings
- Save your favorite prompts for reuse

## Troubleshooting

### Common Issues

**AI Generation Not Working**
- Check your subscription status
- Ensure you have internet connection
- Try a shorter or simpler prompt
- Remove and re-add reference file

**File Upload Fails**
- Check file size limits
- Verify file format is supported
- Check available storage space
- Try a different browser

**Timeline Issues**
- Refresh the page
- Clear browser cache
- Check project is saved
- Verify media files exist

**Export Problems**
- Check export settings match your plan
- Ensure enough storage space
- Verify project has content
- Try exporting at lower quality first
