# MandaStrong Studio - Demo Video Guide

## Accessing the Demo

The demo presentation is a standalone HTML page separate from the main application.

### Local Development
```
http://localhost:5173/demo.html
```

### Production
```
https://yourdomain.com/demo.html
```

## What's Included

The demo page features:

### 1. Hero Section
- Animated title and tagline
- Key statistics (720+ tools, 21 pages)
- Call-to-action buttons
- Smooth scroll navigation

### 2. Features Overview
- 6 feature cards with icons and descriptions

- Visual hierarchy with gradient backgrounds

### 3. Workflow Timeline
- Visual representation of the 21-page journey
- Step-by-step walkthrough
- Animated elements on scroll

### 4. Tech Stack
- 8 technology cards
- Modern stack highlights
- Clean, organized layout

### 5. Pricing Section
- All 3 subscription tiers
- Feature comparisons
- Call-to-action buttons

### 6. Final CTA
- Launch app button
- Contact information
- Professional closing

## Animations & Interactions

- **Fade In Up**: Main titles and hero content
- **Slide In**: Timeline elements from left and right
- **Hover Effects**: Cards scale up and glow on hover
- **Scroll Animations**: Sections fade in as you scroll
- **Smooth Scrolling**: Navigation links smoothly scroll to sections
- **Pulse Glow**: Primary CTA buttons have a pulsing glow effect

## Customization

The demo is built with:
- Tailwind CSS (CDN version for standalone use)
- Pure vanilla JavaScript
- No dependencies on the main app
- Fully responsive design

## Recording as Video

To create an actual video from this demo:

1. **Screen Recording**
   - Use OBS Studio, Loom, or built-in screen recorder
   - Set resolution to 1920x1080 for best quality
   - Record at 60fps for smooth animations

2. **Navigation Tips**
   - Start at the top and slowly scroll through
   - Hover over feature cards to show interactions
   - Demonstrate smooth scrolling between sections
   - End with the final CTA

3. **Voiceover Script** (Optional)
   ```
   "Introducing MandaStrong Studio - the all-in-one movie creation platform.

   With 720+ AI-powered tools spread across 21 intuitive pages, creating
   professional movies has never been easier.

   From AI image and video generation, to professional editing tools, to
   community collaboration - everything you need is in one place.

   Built on modern technology like React, TypeScript, and Supabase, with
   enterprise-grade security and scalability.

   Choose from three flexible pricing tiers and start creating today.

   MandaStrong Studio - Your vision, our platform."
   ```

4. **Background Music**
   - Use royalty-free music from YouTube Audio Library
   - Keep volume low to not overpower voiceover
   - Suggested genre: Cinematic, Inspirational, Modern
